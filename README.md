# Proyecto MoonBot

## Instalación

```
npm i
```

## Configuración

Crear el archivo `.env` dentro del directorio `src` que contenga los siguientes datos:

```
TOKEN=
PREFIX=
OWNERID=
```

Rellenar la información faltante después del `=`, ejemplo:

```
TOKEN=PegaElTokenAqui
PREFIX=m!
OWNERID=280985817097306113
```

## Ejecución

```
cd src
node index.js
```
